## เครื่องมือที่ใช้

 - angular 11
 - node 12
 - bootstrap 4
 - ngx-bootstrap

## setup project ##

 1. npm install ก่อนการใช้งานทุกครั้ง
 2. ng build ทุกครั้งเมื่อมีการแก้ไข lib
